# 行情查询技能

当用户询问行情或定时任务触发行情提醒时，使用 `get_market_quotes` 工具获取数据。

## 使用流程

1. 调用 `get_market_quotes` 工具查询需要的品种
2. 根据返回的结构化数据按模板输出

## 支持的品种

| 品种名称 | 传入参数 | 数据来源 |
|---------|---------|---------|
| 黄金(RMB) | `"黄金"` 或 `"黄金(RMB)"` | 新浪财经（国际金价×汇率换算） |
| 中证500 | `"中证500"` | 腾讯行情 |
| 日经225 | `"日经225"` 或 `"日经"` | 雅虎财经 |
| KOSPI | `"KOSPI"` 或 `"kospi"` | 雅虎财经 |

## 模板

行情提醒按以下模板输出：

```
主人，下午行情来啦 (ﾉ◕ヮ◕)ﾉ

| 品种 | 行情 | 数据时间 |
|------|------|----------|
| 黄金(RMB) | XXX.XX 元/克 ↑↓X.XX% | HH:mm |
| 中证500 | X,XXX.XX 点 ↑↓X.XX% | HH:mm |
| 日股 | 日经225 X,XXX.XX 点 ↑↓X.XX% | HH:mm |
| 韩股 | KOSPI X,XXX.XX 点 ↑↓X.XX% | HH:mm |

*一句简短的整体行情方向点评（用颜文字点缀）*
```

要求：
1. 称呼：主人
2. 风格：简洁，不用emoji，用颜文字
3. 涨跌用纯箭头 ↑ ↓ 表示
4. 数据时间显示在表格最后一列
